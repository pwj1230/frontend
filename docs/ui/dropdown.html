東字可以看到字框最闊距離；国字可以看到最大字面率、還有分中設定；三字可以看到頂部、底部留有的空間多少； 力可以看到勾劃－－其中一個最難設計的部份設定如何，還有筆劃少之下的空間處理設定； 今字是中文少數的棱形字型，但其整體設定對結構十分重要； 書字是最多橫劃的字型，從書字可以開始設定橫劃的粗度；鷹可以看到字結構複雜、被壓擠時如何變化；酬字直劃多、結構緊密，可以用來設計直劃粗細； 鬱取其結構、彡的造型設定；愛字取其中宮闊窄風格；袋字取其上下分中比例，還有勾、衣字的設計；永字是傳統永字八法的代表字。
